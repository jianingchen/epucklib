void el_config_stepper_motor ( el_stepper_motor_param  param,
int  value 
)

configure various parameters of the stepper motor module

Parameters
paramtarget parameter
valuevalue to be applied

Use param to select which paramter to change.

Currently, this function is only used to configure the artificial acceleration of the stepper motors, which are typically used to make the motion of the robot more smooth.

For example, the following code enables a linear artificial acceleration with a rate of 2000:

1 ...
2 el_disable_stepper_motor();
3 el_config_stepper_motor(EL_SPEED_ACC_ENABLE,true);
4 el_config_stepper_motor(EL_SPEED_ACC_LINEAR_TERM,2000);
5 el_enable_stepper_motor();
6 ...

A linear acceleration rate of 2000 means the stepping rate of the motors can reach 1000 from 0 in 0.5 sec. By default, artificial acceleration is disabled while the acceleration rate is set to 3000.
